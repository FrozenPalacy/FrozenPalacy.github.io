高中历史 选必一 要点合集

一、政治制度

1. 中国古代政治制度

  1. 中央政治制度

    夏：世袭制—>商：内外服制—>西周：宗法制、分封制—>春秋时期：贵族等级分封制开始解体—>战国：变法—>秦：皇帝制度、三公九卿制—>西汉：三公九卿制，设中朝—>东汉：尚书台（刘秀）—>隋唐：三省六部—>宋：两府三司—>元：中书省—>明：废中书省和宰相，设内阁—>清：军机处

    战国：郡县制、官僚制逐渐产生

    秦：丞相和诸大臣听命于皇帝。三公指丞相（百官之长、统领众官，丞相府是行政中枢）、太尉（主管军事）、御史大夫（副丞相，最高监察官，协理国政）

    西汉：晚期中朝尚书权力逐渐增大

    隋唐：中书省（决策机构）、门下省（审议政令的机构）、尚书省（执行机构）；三省长官中书令、门下侍中、尚书令与尚书仆射并称宰相；尚书省下设吏户礼兵刑工六部。

    宋：同中书门下平章事（最高行政长官，宰相），枢密院（军政），三司（财政大权）

  2. 地方行政制度

    秦：郡县制—>西汉：郡国并行、郡县（推恩令、刺史）—>东汉：州、郡、县三级制—>隋：州、县二级制—>唐：道、州、县三级制—>宋：路、州（府）、县三级制—>元：行中书省，行省制，省、路、府、州、县—>明清：省、府、县三级制

    秦：分郡县两级，郡县主要管理都由中央直接任命

    唐中期以后：设节度使

    明：巡抚、巡按，总揽一省之权

    清：巡抚成一省长官，设总督，掌管一省或数省军政大权

2. 西方古代和近代政治制度

  1. 古希腊 & 古罗马

    B.C. 8世纪至B.C. 6世纪，古代希腊城邦发展。

    雅典：民主政治，古典民主，国家权力掌握在公民大会、议事会、官员和公民法庭手中。全体成年男性公民组成公民大会。是建立在奴隶制度之上的民主。

    斯巴达：寡头政治。有公民大会是名义上最高权力机关，王位世袭，国王垄断军权；长老会30人终身任职；监察官一年一任。

    希腊陶片放逐法：针对势力足以影响城邦稳定的政治家。放逐十年，中途可以被城邦召回。

    罗马共和国：贵族寡头政治。执政官、元老院、公民大会。

    B.C. 1世纪末，罗马共和国崩溃，屋大维建立元首制，在罗马形成君主政治。

  2. 中古西欧

    1. 封君封臣制

    2. 世俗王权和基督教会权力长期并立。

      14世纪初，法国国王腓力四世召开三级会议，法国进入等级君主制阶段

      13世纪初，英国国王签署**《大宪章》**，议会形成。14世纪，议会分为上下两院。

  3. 西方资本主义政治制度

    1. 17世纪英国资产阶级革命奠定了君主立宪制的基础。
    2. 1775年，独立战争由北美的13个英国殖民地发起，独立战争开始；1776年美利坚合众国成立。1787年制定宪法，三权分立，民主共和，联邦制共和国
    3. 1792年，法国首次建立共和国，共和制度不稳固；1870年法国建立第三共和国，1875年通过新宪法，设立国会（参议院、众议院），经过数年动荡确立了共和制。

3. 中国近代至当代政治制度

  1. ROC

    1. 1912 南京临时政府

      《中华民国临时约法》、主权在民、平等自由原则，行政、立法、司法三权分立

    2. 1912国民党在国会选举中获胜，宋教仁成为国会多数党领袖，威胁袁世凯独裁。1913年春被刺杀。随后国民党发动二次革命，被镇压。

      镇压后袁世凯强迫国会议员选举他为正式大总统，随后解散国民党和国会。

    3. 1915年袁世凯复辟。不久死了。随后北洋军阀内部派系纷争，军阀割据。

    4. 1923年6月国共合作，1926北伐，1927国共合作破裂（四·一二反革命政变，七·一五汪精卫分共）。

    5. 1928年国民党统治，军政时期结束，训政时期开始。五权分立：行政、立法、司法、考试、监察。一党专政。

    6. 1948宪政。

  2. CPC在根据地和解放区

    1. 1930年夏建立起十几个农村革命根据地。
    2. 1931年11月7日至20日，江西瑞金，中华苏维埃第一次全国代表大会召开，通过土地法、劳动法，制定宪法大纲，成立中华苏维埃共和国临时中央政府，毛泽东当选为中央执行委员会主席和中央执行委员会人民委员会主席。
    3. 抗日期间在革命根据地设边区政府，“三三制”（与党外人士民主合作，各级政权机关的工作人员，共产党只占三分之一）。
    4. 解放战争期间在解放区设置行政区，行政区设军政委员会或人民政府。

  3. PRC政治制度（略）

4.中国历代变法改革

  1. 战国秦 商鞅变法：废井田，奖励耕织，奖励军功，实行什伍连坐
  2. 鲜卑族拓跋部北魏孝文帝拓跋宏：迁都洛阳，易服装，改汉姓，说汉话，通婚姻
  3. 1069 宋神宗时期 王安石变法：目的是“富国强兵”，五六年后失败，变法措施被废止，王安石被罢职
  4. 1572 明神宗时期 张居正改革：除一条鞭法外，其他全部在他死后废止
  5. 1898 戊戌变法：康有为、梁启超等维新志士提起，创学会，办报刊，兴学堂，提倡西学，主张学习日本、俄国变革政治。1898年6月起光绪帝颁布一系列变法诏令，因触犯守旧势力利益，9月慈禧太后发动政变，废除变法措施。这一失败证明了资产阶级改良道路在半殖民地半封建社会的中国行不通。
  6. 1901 清末新政：改革教育、派遣留学生、编练新军、振兴商务、奖励实业，促进民族资本主义发展
  7. 1978改开

二、选官制度与管理（选官、考核、监察）

1. 中国古代

  1. 选官制度：

    西周、春秋：世官制（官位世袭）—>秦：以法为教、以吏为师—>汉武帝：察举制—>曹丕（220）：九品中正—>隋唐宋：科举—>元：科举（时断时续）—>明：科举

  2. 秦：向官吏学习律令而为官

  3. 两汉时期积功劳为官和征辟是两汉官员选拔制度的重要补充

  4. 九品中正在后期随着门阀士族势力的发展，中正选人只看家世不看道德才能，形成“上品无寒门，下品无势族”

  5. 隋唐科举分制举（皇帝自设科目考试选人）和常举（每年举行，科目多），以明经和进士两科最受社会重视。考试合格后经吏部选拔才能为官。两宋时期进一步完善。

  6. 明代科举分乡试、会试、殿试三级，乡试考中者称“举人”，会试由礼部主持，考中者称“贡士”，殿试皇帝主持，通过者称“进士”。八股取士。

  7. 考核制度：

    秦汉：上计制—>魏晋南北朝：有考核法规，但流于形式—>隋唐：尚书省吏部—>明：考满、考察—>清：京察、大计

  8. 明：考察包括外地官员三年一次的朝觐考察和京官六年一次的京察，查处官员贪、酷和不作为

  9. 清：京察考察京官，大计考察外省文官

  10. 监察制度：

    秦汉：以御史大夫为首的中央监察体系—>汉武帝：刺史—>隋唐：机构为御史台，机构长官为御史大夫—>宋：承唐制，设御史台，台谏合一—>元：御史台、行御史台、肃政廉访司—>明清：都察院、六科

  11. 唐：唐太宗分全国为十道监察区，委派监察官巡回监察。

  12. 宋：地方划分路为监察区。台谏合一：御史有了谏官的议事权，谏官有了御史的监察权

2. 西方文官制度

  1. 中世纪：国王赐官—>17—18世纪：政党分肥制—>1870英国确立文官制度—>1883美国确立文官制度—>二战后，法、德、日建立文官制度
  2. 背景：工业革命后英国等资本主义国家生产力水平提高，国家管理职能快速扩展，人们教育程度提高，对能迅速处理日常事务的职业官僚体系和人民平等参与政府工作的需求提高。
  3. 过程（英国）：18世纪初，英国规定除大臣外的其他官员不得当选为下院议员。19世纪初为保证政府工作不受政党更替影响，部分政府部门设立常务次官。于是英国政府官员分为政务官和事务官（文官）两类。文官不受党派影响，可以长期任职。1855年在克里米亚战争中英军伤亡惨重，人们对政府的管理能力不满，英国颁布法令建立不受党派干涉的文官委员会，1870年确立文官制度。
  4. 特点：
    1. 符合规定的公民都可以参加文官考试，择优录用
    2. 文官要在政党之间保持中立
    3. 实现政治和管理分离，有利于政府工作稳定性和持续性
    4. 滋生官僚习气和僵化现象，文官人数膨胀，增加国家财政负担
    5. 文官层次变多，出现争权夺利现象，影响工作效率

3. 中国近代

  1. 1905光绪停科举，学堂选官、留学毕业生选官
  2. 1912南京临时政府：文官制度（基本停留在纸面）—>北洋政府：文官考试制度—>南京国民政府：公务员制度—>PRC：干部制度、公务员制度
  3. 北洋政府的文官考试只有民国男子年满21岁者可以应试，女子不得参加考试。国民政府的公务员制度允许女子参加考试。
  4. 考核：北洋政府、南京国民政府：甄别

三、法律与教化

1. 中国古代

  1. 法律

    战国：子产《铸刑书》，中国最早的成文法—>秦：《秦律》—>汉：《九章律》—>魏晋：律令儒家化—>唐：《唐律疏议》—>宋：963年《宋刑统》—>明：《大明律》—>清：《大清律例》

  2. 秦汉时期还有《二年律令》。秦汉朝廷还颁布法律文告，称“令”。

  3. 唐律疏议是中华法系确立的标志，是礼法结合的典范

  4. 明代《大明律》开创了律令合编的体例。

  5. 社会教化

    战国：儒家德治—>秦：法家（焚书坑儒）—>汉：以经注律—>唐：732年《大唐开元礼》—>宋：乡约，《吕氏乡约》—>明：六谕—>清：圣谕，约律合流

  6. 以经注律从汉武帝之后开始

2. 西方近代

  1. B.C.450 罗马共和国《十二铜表法》—>6世纪 东罗马皇帝查士丁尼 《罗马民法大全》（查士丁尼法典）

  2. 《罗马民法大全》是古罗马法律的最高成就和近代西方法律制度的渊源。

  3. 英美法系（普通法系）：11世纪王室设立法院，派法官到各地巡回审判。12世纪，建立在罗马法和习惯法基础上的普通法形成。13世纪《大宪章》确立法律至上、王权有限原则。光荣革命后确立君主立宪制。美国等国家学习英国法律，构成普通法系aka英美法系。

  4. 大陆法系（民法系）：1789年法国大革命后制定了一系列法律，**1804年拿破仑颁布《法国民法典》**aka《拿破仑法典》，与在此前后颁布的法律一起构成了法国成文法体系，最终确立了法国的资产阶级法律制度。后来逐渐形成以罗马法为基础、《法国民法典》为代表的大陆法系。

  5. 特征

    1. 在国家权力结构层面上坚持权力制衡、三权分立。
    2. 坚持程序公正和无罪推定。
    3. 建立了律师制度和陪审团制度。
    4. 注重保护个人权利，确定了私有财产制度。

  6. 392年，基督教成为罗马国教。476年西罗马帝国灭亡，在帝国废墟上建立的日耳曼人国家逐渐接受了基督教。基督教的宗教伦理和教化作用强化了教会对人们的控制，深刻影响了人们的思想意识和日常行为。

  7. 宗教改革后，西欧基督教分裂为天主教和新教。

3. 中国当代

  1. 20世纪五十年代，社会主义法制初创。
  2. 2020.5，十三届全国人大三次会议通过《中华人民共和国民法典》，社会生活的百科全书，在法律体系中居于基础性地位。
  3. 教化：社会主义革命和建设时期：英勇奋斗的革命传统和艰苦奋斗的精神—>改开：加强社会主义精神文明建设，“五讲四美三热爱”—>1994把爱国主义教育作为加强精神文明建设的基础工程加以推进—>2006中共十六届六中全会提出建设社会主义核心价值体系的战略任务—>2012中共十八大形成社会主义核心价值观
